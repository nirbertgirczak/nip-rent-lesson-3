# NIP rent

Aplikacja do rozliczania najmowanych mieszkań.

## Getting started

- Używając *Python3.12* utwórz *venv* i zainstaluj zależności

```sh
python3.12 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
